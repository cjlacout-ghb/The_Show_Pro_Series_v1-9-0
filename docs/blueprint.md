# **App Name**: Softball Showdown Central

## Core Features:

- Team Roster Input: Input the names of the 6 participating teams: ACCIN VORTEX (ARG), CACIQUES BY SWING (CHI), CITY PAN (CHI), MAYO'S (MEX), SOUTH ARGENTINA (ARG), and TEAM FINCA JUJURE (VEN).
- Preliminary Round Schedule and Results: Input the game schedule and results for the preliminary round, with 15 blank spaces for each game.
- Championship Game Input: Input the result for the championship game.
- Standings Display: Display the standings in a tabular format with columns for Position (POS), Team Name (TEAM), Won (W), Lost (L), Runs Scored (RS), Runs Against (RA), W-L Percentage (PCT), and Games Behind First (GB).

## Style Guidelines:

- Default Theme: Dark mode with a dark blue background (#1E293B).
- Primary Color: White (#FFFEFB) for text and primary interactive elements.
- Secondary Colors: Gold (#FFC000) and Red (#FF3D3D) for rings, highlights, or notifications.
- Use clear and legible sans-serif fonts optimized for readability. Recommended font: 'Inter' sans-serif for headlines and body.
- Clean and structured layout with distinct sections for team input, score entry, and standings.
- Subtle transitions and animations on data updates to provide visual feedback and enhance user experience.
